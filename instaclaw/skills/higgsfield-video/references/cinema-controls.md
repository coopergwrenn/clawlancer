# Cinema Controls Reference

## Camera Bodies
Use in prompts: "Shot on [camera]"

| Camera | Character |
|--------|-----------|
| ARRI Alexa 35 | Gold standard. Rich color, cinematic texture. Best for drama, narrative. |
| RED V-RAPTOR | Ultra sharp, high dynamic range. Best for action, VFX-heavy shots. |
| Sony VENICE 2 | Natural skin tones, gentle highlights. Best for intimate scenes. |
| Blackmagic URSA Mini Pro 12K | Detailed textures, wide latitude. Best for landscapes. |
| Canon EOS C500 Mark II | Warm tones, video-friendly. Best for documentary style. |

## Lenses
Use in prompts: "Shot with [lens] at [focal length]"

| Lens Series | Character |
|-------------|-----------|
| Cooke S7/i | Warm, gentle flare, "Cooke Look." Classic cinema. |
| Zeiss Supreme Prime | Clinical sharpness, neutral. Modern cinema. |
| Canon CN-E | Warm highlights, pleasant bokeh. Broadcast/cinema crossover. |
| Panavision Primo 70 | Organic, dreamy. Classic Hollywood feel. |
| Sigma Cine | Sharp edge-to-edge, value option. |
| Leica Summilux-C | Micro-contrast, 3D pop. Prestige look. |

## Focal Lengths

| Focal Length | Use Case |
|-------------|----------|
| 14-18mm | Extreme wide. Landscapes, architecture, establishing shots. |
| 24mm | Wide. Room interiors, group shots, environmental portraits. |
| 35mm | Standard wide. Walking shots, natural perspective. |
| 50mm | Standard. Most natural to human eye. Dialogue, mid-shots. |
| 85mm | Portrait. Flattering faces, shallow DOF, interview B-roll. |
| 135mm | Telephoto. Compressed backgrounds, intimate close-ups. |
| 200mm+ | Extreme telephoto. Surveillance, compressed landscapes. |

## Aperture

| Aperture | Effect |
|----------|--------|
| f/1.2-1.4 | Ultra shallow DOF. Subject isolation, dreamy bokeh. |
| f/2 | Shallow DOF. Cinematic standard for close-ups. |
| f/2.8 | Moderate DOF. Balanced, common for dialogue scenes. |
| f/4-5.6 | Deep-ish focus. Good for two-shots and medium groups. |
| f/8-11 | Deep focus. Landscapes, architecture, everything sharp. |
| f/16+ | Maximum depth. Rarely used — diffraction softening. |

## Camera Movements
Use in prompts: "[movement type] shot"

| Movement | When to Use |
|----------|-------------|
| Static/locked off | Tension, observational, tableaux |
| Dolly in | Building intensity, drawing viewer in |
| Dolly out | Reveal, pulling back from intimacy |
| Crane/jib up | Reveal scope, transition to wide, uplift |
| Crane/jib down | Coming down to ground level, grounding |
| Steadicam tracking | Following subject through space |
| Handheld | Urgency, documentary feel, chaos |
| Pan (left/right) | Following action, surveying environment |
| Tilt (up/down) | Revealing height, power dynamics |
| Zoom (in/out) | Dramatic emphasis (Hitchcock zoom, crash zoom) |
| Orbital/360 | Circling subject, disorientation, hero moment |
| Whip pan | Quick transition, energy, surprise |

## Lighting Keywords

| Term | Effect |
|------|--------|
| Golden hour | Warm, directional, long shadows |
| Blue hour | Cool, ethereal, pre-dawn/post-sunset |
| Chiaroscuro | Dramatic light/shadow contrast |
| Rembrandt lighting | Triangle of light on cheek, classic portrait |
| High-key | Bright, even, minimal shadows |
| Low-key | Dark, moody, heavy shadows |
| Practical lighting | Motivated by in-scene sources (lamps, windows) |
| Neon | Cyberpunk, urban night, color contrast |
| Volumetric | Visible light rays, fog, atmosphere |

## Film Stock / Look Keywords

| Term | Effect |
|------|--------|
| Film grain | Organic texture, analog feel |
| Anamorphic | Oval bokeh, horizontal lens flares, wider aspect |
| Kodak Ektachrome | Warm, saturated, contrasty reversal film look |
| Fuji Velvia | Ultra-saturated, vivid colors |
| Kodak Vision3 500T | Tungsten-balanced, cinema standard |
| Bleach bypass | Desaturated, high contrast, gritty |

## Prompt Template

```
[SUBJECT AND ACTION]. Shot on [CAMERA BODY], [LENS] [FOCAL LENGTH] at [APERTURE], [CAMERA MOVEMENT], [LIGHTING], [FILM STOCK/LOOK].
```

Example:
```
A lone cowboy rides through a desert canyon at sunset. Shot on ARRI Alexa 35, Cooke S7/i 50mm at f/2, steadicam tracking shot, golden hour lighting, Kodak Vision3 500T film stock, subtle film grain.
```
