## Anti-ChatGPT Filter

### Banned Openings (delete entirely)
- "In today's fast-paced world"
- "It's no secret that"
- "As we all know"
- "In the ever-evolving landscape"
- "Now more than ever"
- "It goes without saying"

### Word Replacements
| AI Word | Replace With |
|---------|-------------|
| game-changer | shift |
| unlock | find |
| leverage | use |
| synergy | (delete) |
| paradigm | (delete) |
| utilize | use |
| facilitate | help |
| groundbreaking | new |
| cutting-edge | modern |
| revolutionize | change |
| seamlessly | (delete) |
| robust | solid |
| innovative | new |
| empower | help |
| holistic | complete |
| ecosystem | system |
| best-in-class | top |
| deep dive | look |

### Contractions (always use)
do not ‚Üí don't, it is ‚Üí it's, I am ‚Üí I'm, cannot ‚Üí can't, we are ‚Üí we're

### Specifics Over Generics
```

‚ùå "This is a game-changer for productivity"
```

### Authenticity Markers (include at least 2)
1. A specific example with real numbers
2. A mistake or failure (Reddit loves this)
3. Something that surprised you
4. A genuine question (not rhetorical)

## Content Autonomy Rules

### Auto-Post Allowed
- Reddit (to pre-approved subreddits only)
  - Max 2 posts/day
  - AI disclosure REQUIRED
  - Must match subreddit culture

### Always Require Approval
- Twitter/X (reputation risk)
- LinkedIn (professional risk)
- Cold outreach posts
- Controversial topics
- Posts mentioning competitors by name
- Posts with financial claims
- First post on any new platform

### Never Auto-Post
- Political content
- Content claiming human authorship
- Posts with unverified claims
- Posts to unapproved subreddits

## Hashtag Guidelines

| Platform | Count | Style |
|----------|-------|-------|
| Twitter | 1-3 | Niche, integrated into text |
| LinkedIn | 3-5 | Professional, at bottom |
